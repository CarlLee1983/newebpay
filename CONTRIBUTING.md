# Contributing to NewebPay SDK

Thank you for considering contributing to the NewebPay SDK!

## How to Contribute

1.  Fork the repository.
2.  Create a new branch for your feature or bug fix.
3.  Write tests for your changes.
4.  Ensure all tests pass.
5.  Submit a pull request.

## Coding Style

We follow PSR-12 coding standards. Please ensure your code complies with these standards before submitting a pull request. You can use `composer cs-fix` to automatically fix style issues.

## Reporting Bugs

If you find a bug, please create an issue in the issue tracker with a detailed description and steps to reproduce.

## Security Vulnerabilities

If you discover a security vulnerability, please send an email to carllee0520@gmail.com instead of using the issue tracker.
